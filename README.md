# Downtime Alert

*Downtime Alert* is a web application to monitor your website‚Äôs downtime. Downtime Alert will help you to keep monitoring your website status and inform you immediately if your website goes down. It is also important to keep user's trust by making your website secure using SSL. Downtime Alert will help you by informing you before your SSL and domain get expired so that you can focus on your work.

## Installation
In the root project directory run this command
```
yarn install
```
Run this in the project_root/api
```
composer install
```
## Usage
